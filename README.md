# Chat-App  
Chat App made by using nodejs and socket.io   
Make sure to install nodejs on your computer if npm command is not recognized by your computer it  
To run the app:  
1. npm init -y
2. npm i nodejs
3. npm i socketio
4. npm i express
5. npm i uuid
6. node server.js
   if you have the packages installed step 6 will be enough.
